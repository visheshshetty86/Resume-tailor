# Resume Tailor Frontend

Vite + React + Tailwind frontend for uploading a resume PDF, entering a job listing URL, and preparing a tailored resume flow.

## Run locally

```bash
npm install
npm run dev
```

## Structure

- `src/components/layout` for app shell and header
- `src/components/resume` for resume upload UI
- `src/components/job` for job URL input
- `src/components/action` for primary actions
- `src/components` for shared panels and previews
- `src/hooks` for local UI state
